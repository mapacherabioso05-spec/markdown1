# Práctica UT3-A2 Markdown

## 1. Introducción

Este archivo README.md forma parte de la práctica UT3-A2-Markdown del módulo de entornos de desarrollo.

El objetivo de esta actividad es utilizar diferentes elementos del lenguaje Markdown dentro de un repositorio de GitHub. 

### 1.1 ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite dar formato a textos de una forma sencilla.

## 2. Lista ordenada

Pasos realizados en esta práctica:

1. Crear un repositorio en Github llamado "markdown1".
2. Crear o modificar el archivo "README.md".
3. Añadir diferentes elementos Markdown. 
4. Insertar imágenes locales y externas. 
5. Subir los cambios al repositorio remoto

## 3. Lista desordenada

Elementos utilizados en este archivo: 

- Encabezado. 
- Listas. 
- Tablas.
- Imágenes. 
- Enlaces. 
- Citas.
- BLoques de código.

## 4. Tabla

| Elemento Markdown | Símbolo utilizado | Función |
|---|---|---|
| Encabezado | "#" | Crear Títulos |
| Lista ordenada | "1" | Crear listas numeradas |
| Lista desordenada | "-" | Crear listas con viñetas |
| Enlace | "[Texto](url)" | Acceder a una página web |
| Imagen | "![Texto](ruta)" | Insertar imágenes |
| Cita | ">" | Resaltar una frase o comentario |

## 8. Código Python

Ejemplo de código de Python con resaltado de sintaxis: 

```python
class Persona: 
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
	print("Hola, me llamo", self.nombre)

persona1 = Persona("María", 20)
persona1.saludar()
```
